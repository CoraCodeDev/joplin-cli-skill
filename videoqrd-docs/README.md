# VideoQrd

QR codes for videos. Create beautiful QR codes that link to YouTube/Vimeo videos — perfect for printing, scrapbooks, and sharing.

## What is VideoQrd?

VideoQrd lets you:
1. Paste a YouTube or Vimeo URL
2. Generate a QR code pointing to your video
3. Download the QR code to embed in photos or print
